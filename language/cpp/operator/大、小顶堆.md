### 大顶堆
> queue中的 priority_queue实现
```C++
// 定义
priority_queue<int> maxHeap;
maxHeap.push(1);
maxHeap.push(5);
maxHeap.push(3);
cout << maxHeap.top() << endl;
maxHeap.pop();
cout << maxHeap.top() << endl;
```

### 小顶堆
```C++
priority_queue<int, vector<int>, std::greater<int>> minHeap;
minHeap.push(9);
minHeap.push(1);
minHeap.push(2);
cout << minHeap.top() << endl;
minHeap.pop();
cout << minHeap.top() << endl;
```
